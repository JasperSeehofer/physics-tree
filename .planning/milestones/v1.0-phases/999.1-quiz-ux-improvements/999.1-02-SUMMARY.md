### Task 2: Formula Preview Brace-Stripping Fix

Removed `prop:value=move || input_value.get()` from the formula input element. This two-way binding caused the Leptos reactive system to re-set the DOM input value on each reactive cycle, which mangled structured LaTeX characters (`{`, `}`, `_`) between keystrokes. Without the binding, the browser's native input element owns the value; `on:input` still reads it via `event_target_value` and updates the `input_value` signal for the Check button handler.
