**Imports:**
```python
from dataclasses import dataclass, field
```

**Data models** (per D-01):
- `CheckStatus(Enum)` with PASS="PASS", FAIL="FAIL", WARNING="WARNING"
- `CheckResult` dataclass with `name: str`, `status: CheckStatus`, `detail: str = ""`
- `GateReport` dataclass with `node_slug: str`, `mechanical: list[CheckResult]`, `judgment: list[CheckResult]`, `generated_at: str` (auto ISO timestamp), property `overall_pass` returns True only if no FAIL in mechanical+judgment

**Helper: _strip_frontmatter(text: str) -> str** (per Research Pitfall 2):
- Find lines matching `---`, take content after second `---`
- If fewer than 2 `---` found, return original text

**Helper: _check_latex_balance(phase_text: str, phase_num: int) -> CheckResult:**
- Strip frontmatter first via _strip_frontmatter
- Remove `$$` sequences, count remaining `$` — odd count = FAIL
- Count `\[` vs `\]` — mismatch = FAIL
- Return CheckResult with name `latex_balance_phase_{phase_num}`

**Helper: _check_word_count(phase_text: str, phase_num: int, min_words: int = 50) -> CheckResult:**
- Strip frontmatter first
- Count words via `len(body.split())`
- Below min_words = FAIL with detail showing count and minimum
- Return CheckResult with name `word_count_phase_{phase_num}`

**Helper: _check_prerequisite_existence(node_dir: Path, project_root: Path) -> CheckResult:**
- Read `node.yaml` from node_dir using `yaml.safe_load()` (NEVER `yaml.load()` — per ASVS V5)
- Extract `prerequisites` list
- For each prerequisite slug, check if `content/` tree contains a directory with that name (use `list(content_dir.rglob(slug))`)
- Missing prerequisites = FAIL with detail naming which slugs are missing
- No prerequisites field = PASS (not all nodes require prereqs)
- Return CheckResult with name `prerequisite_existence`

**Function: run_mechanical_checks(node_dir: Path, project_root: Path | None = None) -> list[CheckResult]** (per D-03):
- Resolve project_root via resolve_project_root() if None
- Call `validate_node(node_dir, project_root)` — empty list = PASS CheckResult named "rust_validator", non-empty = FAIL with errors joined
- For each `phase-N.md` file in node_dir (N=0..6): run `_check_latex_balance` and `_check_word_count`
- Run `_check_prerequisite_existence`
- Return all CheckResults

**Function: run_judgment_checks(staging_dir: Path) -> list[CheckResult]** (per D-04):
- Look for `review-report.md` in staging_dir
- If missing: return `[CheckResult("review_report_present", WARNING, "review-report.md not found")]`
- Read file, call `parse_dimension_results(text)`
- If parse returns empty list: return `[CheckResult("review_report_content", WARNING, "No dimensions found")]`
- Map each `DimensionResult` to `CheckResult`: name = `dim.dimension.lower().replace(" ", "_")`, status = `CheckStatus(dim.status.value)`, detail = `dim.feedback`
- Wrap parse in try/except: on error return WARNING CheckResult

**Function: run_gate(staging_dir: Path, project_root: Path | None = None) -> GateReport** (per D-01):
- `mechanical = run_mechanical_checks(staging_dir, project_root)`
- `judgment = run_judgment_checks(staging_dir)`
- Return `GateReport(node_slug=staging_dir.name, mechanical=mechanical, judgment=judgment)`

**Function: write_gate_report(report: GateReport, staging_dir: Path) -> Path** (per D-10, D-11):
- Render Markdown with this exact structure:
  - `# Quality Gate Report: {node_slug}`
  - `**Generated:** {timestamp}`
  - `**Overall:** PASS|FAIL`
  - `---`
  - `## Mechanical Checks`
  - Table: `| Check | Status | Detail |` with one row per mechanical check
  - `## Judgment Checks`
  - Table: `| Check | Status | Detail |` with one row per judgment check
  - `---`
  - `## Full Review Report`
  - Read and append `review-report.md` from staging_dir if it exists, otherwise `_No review report available._`
- Write to `staging_dir / "quality-gate-report.md"`
- Return path

```python
elif args.command == "gate":
    from .quality_gate import run_gate, write_gate_report
    from .staging import StagingManager
    from .config import load_config
    from .subprocess_tools import build_binaries, resolve_project_root
    config = load_config(args.config)

    build_binaries(project_root)
    staging = StagingManager()

    print(f"[gate] Overall: {'PASS' if report.overall_pass else 'FAIL'}")

        print(f"  [{check.status.value}] {check.name}" + (f" — {check.detail}" if check.detail else ""))
```
