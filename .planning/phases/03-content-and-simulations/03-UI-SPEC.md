## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding, inline spacing |
| sm | 8px | Compact element spacing, quiz answer option padding |
| md | 16px | Default element spacing, content column internal padding |
| lg | 24px | Section padding, panel header padding (matches existing `p-6`) |
| xl | 32px | Layout gaps between major content sections |
| 2xl | 48px | Major section breaks (Motivation → Derivation → Intuition transitions) |
| 3xl | 64px | Page-level vertical breathing room at top/bottom of content column |

Exceptions:
- Simulation canvas touch target for Play/Pause: minimum 44px height (accessibility)
- TOC sidebar: 240px fixed width (left rail, does not follow 8-point scale)
- Content column: 700px max-width centered (reading-width constraint, not spacing)
- Simulation breakout width: 960px max-width (wider than content column, centered)

Source: 8-point default scale; exceptions derived from D-03, D-04, D-10 in CONTEXT.md.

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 16px | 400 (regular) | 1.6 | Motivation, intuition, example paragraphs, misconception reveal text |
| Label | 14px | 700 (bold) | 1.4 | Badge labels, TOC links, quiz option labels, section headers in TOC |
| Heading | 20px | 700 (bold) | 1.3 | Section headings (Motivation, Derivation, Intuition, etc.), quiz section titles |
| Display | 28px | 800 (extrabold) | 1.2 | Concept title at top of content page |

Notes:
- Nunito 800 (extrabold) is available (confirmed in style/main.css) — use for display only
- Math blocks (KaTeX): rendered as inline or block; KaTeX inherits Nunito for surrounding text, math glyphs use KaTeX's own Computer Modern WOFF2
- Code blocks (if any): monospace fallback `ui-monospace, monospace` at 14px weight 400
- LaTeX display equations: block-level, centered, 20px equivalent size via KaTeX CSS

Source: existing panel.rs uses `text-xl font-bold` (20px/700), `text-base` (16px), `text-sm` (14px); Nunito 800 declared in main.css. Display size derived from D-01 (full dedicated content page warrants larger title).

## Interaction Contracts

### TOC Scroll-Spy

- Section headings have `id` attributes matching TOC link `href`s
- Active section: `text-leaf-green font-bold` (was `text-mist` inactive)
- Intersection observer threshold: 0.4 (section is 40% visible = active)
- Smooth scroll on TOC click: `scroll-behavior: smooth`

### Derivation Stepper

- Initial state: Step 1 visible, Steps 2–N hidden
- "Next step" button: `bg-bark-mid hover:bg-bark-light text-petal-white text-sm font-bold rounded px-4 py-2`
- Completed step: shows green checkmark icon (inline SVG, `text-leaf-green`), step text remains visible
- Transition: new step fades in (`opacity-0 → opacity-100`, 200ms ease-out)
- Each step layout: LaTeX block on top, plain-language explanation below in `text-mist`

### Misconception Card

- Initial state: shows misconception statement in `text-sun-amber font-bold`, "Tap to reveal" hint in `text-mist text-sm`
- After click: reveals explanation text in `text-petal-white`; card background transitions `bark-mid → bark-light`
- Card border: `border border-bark-light rounded-lg` (matches existing card pattern)

### Quiz Checkpoint (Soft Block)

- Content below checkpoint: `opacity-40 blur-[2px] pointer-events-none` until answered or skipped
- "Skip" button: `text-sm text-mist underline hover:text-petal-white` (not a primary action)
- On correct answer: green border flash on selected option, blur removed with 300ms transition
- On first wrong answer: hint text appears in `text-sun-amber text-sm`, no border change yet
- On second wrong answer: correct answer option gets `border-bloom-pink` border + explanation reveals

### Multiple Choice Option

- Default: `bg-bark-mid border border-bark-light rounded-lg p-3 text-base text-petal-white cursor-pointer hover:bg-bark-light`
- Selected (before submit): `border-nebula-purple bg-bark-light`
- Correct: `border-leaf-green bg-bark-mid`
- Wrong: `border-bloom-pink bg-bark-mid`

### Formula Input

- Input field: `bg-bark-mid border-2 border-nebula-purple rounded px-3 py-2 text-petal-white font-mono text-sm`
- KaTeX preview renders below the input in real-time as user types
- Submit: "Check formula" button in `bg-leaf-green text-void font-bold` (same CTA pattern as primary button)

### Simulation Play State

- Paused (initial): large centered play button overlay (48px circle, `bg-leaf-green`, play icon in `text-void`)
- Playing: overlay hidden; pause button in control row
- Reset: returns to initial static state, clears guided challenge status

### Concept Page Navigation

- "Back to graph" link: top-left, `text-sm text-mist hover:text-petal-white` with left-arrow inline SVG (matches panel.rs back button pattern)
- "Next concept" card at bottom: `bg-bark-dark border border-bark-light rounded-lg p-6` with concept title, brief description, `→` arrow
