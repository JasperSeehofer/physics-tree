| Property | Value |

## Spacing Scale

Declared values (all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, node label internal padding |
| sm | 8px | Tooltip padding, badge margin |
| md | 16px | Panel section padding, search input padding |
| lg | 24px | Panel header padding, section breaks |
| xl | 32px | Panel width inner padding at wide breakpoint |
| 2xl | 48px | Panel slide-in offset from viewport edge |
| 3xl | 64px | Reserved for page-level layout (not used in this phase) |

Exceptions:
- Graph canvas: fills 100% of available viewport height — no fixed spacing token applies
- Node touch/click targets: minimum 44x44px regardless of rendered node size (accessibility minimum)
- Panel width: 320px collapsed, 400px at viewport ≥ 1280px — not a spacing token, a layout dimension

## Typography

All text uses Nunito. Exactly 4 sizes, 2 weights in use.

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 16px | 400 (regular) | 1.5 | Panel description text, prereq list items |
| Label | 14px | 700 (bold) | 1.4 | Node labels on graph, tooltip concept title, badge text, search result items |
| Heading | 20px | 700 (bold) | 1.2 | Panel concept title |
| Display | 28px | 700 (bold) | 1.1 | Reserved — not rendered in this phase (carried from Phase 1 landing) |

Source notes:
- Font family from style/main.css `--font-sans: "Nunito"`
- Weights 400 and 700 are the declared weights for this phase. Weight 800 (extrabold) is loaded in style/main.css but is not used in this phase — it will be introduced when Display text is first rendered.
- 14px label is the minimum rendered size — never smaller on graph canvas

## Color

All tokens already defined in `style/main.css @theme`. No new tokens needed.

| Role | Token | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | `--color-void` | #0d0f14 | Graph canvas background, page background |
| Secondary (30%) | `--color-bark-dark` / `--color-bark-mid` | #1a1d24 / #252932 | Top nav bar, right panel background, tooltip background |
| Accent (10%) | `--color-leaf-green` | #4caf7d | Selected node highlight ring, active prereq chain edges, search result active item |
| Prereq path glow | `--color-sun-amber` | #f4b942 | Highlighted prerequisite chain edges only (thickened, amber glow) |
| Underground roots | `--color-nebula-purple` | #8b5cf6 | Mathematical foundation nodes and edges below ground line only |
| Destructive | `--color-bloom-pink` | #e8547a | Not used in this phase — no destructive actions |

Accent reserved for:
- Selected node highlight ring (2px ring, leaf-green)
- Active prerequisite chain edges when a node is selected
- Search result active/hover item background (leaf-green at 20% opacity)
- "Selected" state indicator in right panel header

Node depth-tier color mapping (within botanical metaphor, uses existing tokens):
- Root nodes (depth_tier 0–1): `--color-bark-light` fill (#2e3340), `--color-nebula-purple` stroke — underground or near-ground
- Trunk nodes (depth_tier 2–3): `--color-bark-mid` fill, `--color-petal-white` label
- Branch nodes (depth_tier 4–5): `--color-bark-mid` fill, `--color-leaf-green` stroke
- Leaf nodes (depth_tier 6+): `--color-leaf-green` fill at 70% opacity, `--color-petal-white` label

Ground line: horizontal gradient band — `--color-bark-dark` fading to transparent, 24px tall, at y-position where depth_tier 0 nodes sit.

Unselected node dim: all non-highlighted nodes dim to 30% opacity when a node is selected (source: CONTEXT.md decision).

Text colors:
- Primary text: `--color-petal-white` (#f0f2f5)
- Secondary / muted text: `--color-mist` (#8892a4) — NodeType badges, depth tier labels, disabled CTA

## Component Inventory

All components are Leptos 0.8 RSX. No external component library.

### GraphCanvas
- Full-viewport canvas element hosting Sigma.js WebGL renderer
- Receives node/edge position data from Web Worker via postMessage
- Manages Sigma.js instance lifecycle via wasm-bindgen JS interop
- Emits: node-clicked, node-hovered, node-hover-cleared events to Leptos reactive signals

### NodeTooltip
- Appears on node hover, 200ms delay, disappears on mouse-leave
- Position: offset 12px above cursor (not above node center — avoids obscuring adjacent nodes)
- Contents: concept title (14px bold), NodeType badge (12px, `--color-mist` background)
- Dismissed automatically when node is clicked (panel opens instead)
- Max width: 200px, wraps if needed
- Background: `--color-bark-mid` at 95% opacity, 8px border-radius (`--radius-card`)

### RightPanel
- Slides in from right: 320px wide (400px at ≥ 1280px viewport)
- Graph canvas shrinks to accommodate (CSS flex layout, panel takes fixed width)
- Slide animation: 200ms ease-out translate from +320px to 0
- Header: concept title (20px bold), NodeType badge, branch name, depth tier label
- Body sections: Description (body text), Prerequisites (flat list), Metadata
- Footer: "Learn this concept" button — disabled, `--color-mist` text, `cursor-not-allowed`, tooltip on hover: "Coming in Phase 3"
- Dismiss: X button top-right (24px touch target, sm padding) OR clicking outside panel area on canvas
- Back navigation: back arrow button appears after first navigation — returns to previous node selection

### SearchInput
- In existing top nav bar (slot: right side of nav, left of user menu area)
- Input width: 240px at rest, expands to 360px on focus (CSS transition 150ms)
- Typeahead dropdown: appears below input, max 6 results, `--color-bark-mid` background
- Each result row: concept title (14px bold), branch label (12px, `--color-mist`), 40px row height
- Keyboard: arrow keys navigate results, Enter selects, Escape clears
- Debounce: 200ms after last keystroke before query fires

### EdgeRenderer (Sigma.js custom renderer)
- prerequisite edges: solid line, 1.5px, `--color-bark-light`
- derives_from edges: dashed line (4px dash, 4px gap), 1.5px, `--color-bark-light`
- applies_to edges: dotted line (2px dot, 4px gap), 1.5px, `--color-bark-light`
- mathematical_foundation edges: double line (two parallel 1px lines, 3px gap), `--color-nebula-purple`
- Highlighted prereq chain edges: 3px, `--color-sun-amber`, glow effect (drop-shadow 0 0 4px amber)

## Layout Contract

### Viewport structure
- Top nav: 56px tall, `--color-bark-dark` background, full width, sticky
- Canvas area: `calc(100vh - 56px)`, full width, flex row
- Right panel: 320px (or 400px at ≥ 1280px), slides in from right when node selected
- Graph canvas: fills remaining width when panel is open (flex: 1)

### Minimum supported viewport
- Desktop: 1024px wide minimum (per REQUIREMENTS.md ACCT-04 desktop+tablet)
- No horizontal scrolling at 1024px with panel open (graph canvas min-width: 640px)

### Initial camera position
- Graph loads fully zoomed out to show entire forest/grove
- All trees visible in initial viewport — no scrolling needed to orient
- User zooms in to explore (source: CONTEXT.md "Overview initial camera" decision)
